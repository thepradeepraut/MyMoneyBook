# MyMoneyBook (Android APK)
# This is an early **BETA testing version**.  
> Some features may change or improve based on user feedback.  
> Please report any bugs, issues, or suggestions!
MyMoneyBook is a personal finance and expense tracking app.

- Track incomes & expenses by category
- Manage multiple accounts (cash, bank, wallet)
- View budgets, categories, and transaction history
- Built with Node.js(Vite + Supabase) and wrapped as a native Android app using Capacitor

## How to install

1. Download the APK to your Android phone.
2. Open the APK file.
3. If asked, allow installation from **Unknown sources** / **This source**.
4. Tap **Install** and wait until it finishes.
5. Open **MyMoneyBook** from your app drawer.
